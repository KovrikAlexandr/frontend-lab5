О проекте

VinylHub — магазин винила

Стек

  • Frontend: Vite, SCSS, Fetch API, ES-модули
  • Backend: Node 18, Express, PostgreSQL, pg
  • Инфраструктура: Docker, Docker Compose, .env

Архитектура

MPA с отдельными страницами каталога, карточки товара, корзины и оформления. Общий JS-модуль и сервисы для API, корзины и локального хранилища.

API

REST: список товаров, товар по id, создание заказа. Формат JSON, валидация на backend.

Дальше

Авторизация, админ-панель, платежи, интеграции с внешними API.

VinylHub

Онлайн-магазин винила и подборка культовых релизов.
